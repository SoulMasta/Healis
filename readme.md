# Healis

**Healis** — это платформа, упрощающая взаимодействие студентов с университетской инфраструктурой.  
Приложение предоставляет единый интерфейс для доступа к расписанию занятий, учебным материалам и академическим сервисам.

Основная цель проекта — сделать доступ к образовательной информации быстрым, удобным и безопасным.

---

# Назначение системы

Healis разрабатывается как внешнее приложение, интегрируемое с информационными системами университетов.

Платформа позволяет студентам:

- просматривать расписание занятий
- получать доступ к учебным материалам
- централизованно работать с академической информацией
- использовать единый интерфейс вместо множества университетских систем

---

# Основные возможности

- просмотр расписания занятий

# Интеграция с университетами

Healis может интегрироваться с университетскими системами для получения академических данных.

Интеграция предусматривает:

- аутентификацию через SSO университета
- получение данных через API
- режим доступа **read-only**

Система **не изменяет данные** в информационных системах университета.
Healis является внешним сервисом и не требует прямого доступа к базам данных университетских систем.

# Безопасность

В системе реализованы базовые механизмы обеспечения информационной безопасности:

- защищённые каналы передачи данных (HTTPS / TLS)
- аутентификация пользователей через SSO
- разграничение прав доступа
- журналирование действий пользователей
- минимизация обрабатываемых данных
